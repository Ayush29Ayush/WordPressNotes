1. There are 3 phases to a Wordpress project:
a) Phase - 1 : Foundation
b) Phase - 2 : Development and Designing
c) Phase - 3 : Enhancement
** Refer the Overview.png image for more details.

2. The left panel in the admin page is the most important which contains:
a) Dashboard - The general home page.
b) Posts - The page that contains all the posts.
c) Media - The page that contains all the media.
d) Pages - The page that contains all the pages.
e) Comments - The page that contains all the comments.
f) Appearance - The page that contains all the appearance settings like "Themes" and "Editor"
g) Plugins - The page that contains all the installed plugins and their settings.
h) Users - The page that contains all the users.
i) Tools - The page that contains all the tools like "Import", "Export", "Site Health Check", etc.
j) Settings - The page that contains all the settings like "General", "Reading", "Writing", "Discussion", etc.

3. Go to the settings page and click on "General". Set the site title and tagline. Save the changes.
4. Go to Appearance page and click on "Themes", then "Add new Theme", then search for "Astra" and install it. Activate the theme.
5. Go to Plugins page and search for "Starter Templates" and "Templately", then install it. Activate the plugins.
6. These 2 plugins will provide you various options to choose a theme of your choice.
7. Now go to Appearance page and click on "Starter Templates" and fill in the details as per your requirements.
8. Click on the builder on the top right side and select "Elementor". Save the changes.
9. Now you will also have a "Templates" option in the left panel.
10. From the dropdowns, select "Plumber" and select "Local Business" template. Clicking the template opens a preview page where you can make minor changes before installing it. Click on continue. Fill in basic details and click on "Submit and Build My Website". Upon successful completion, you will get these message:

13. Now visit the page and click on "Edit with Elementor". View all the options and make changes as per your requirements. Save the changes.
14. The Benifit of Elementor is that you have to just click on a selction from your website and the edit functionalities related to it will be displayed.

16. When you get these options, they are distributed into 3 categories
a) Content -> To change the content of the page like text, HTML Tag, etc.
b) Style -> To change the style of the page like alignment, typography, text color, text shadow, etc.
c) Advanced -> To change the advanced features of the page like margin, paddingg, width, Z-index, motions, responsiveness, etc.

17. Use "Ctrl+Z" to undo and "Ctrl+Y" to redo.
18. Add "Link" to a button and click on the "->" button to open Icon Library and select any icon from the library. Save the changes.
19. If after the changes, the element is not looking good and you want to revert the changes, right click on the button and select "Reset Style", this will revert all the changes for that element to the global settings.
20. Explore each option from the Style and Advanced section. 

21. Get high quality images from "https://unsplash.com" and use them in your website.
22. Do not directly download the image with its default size as it may be high in size and this will increase the load time of your website, instead download the medium sized image which will be smaller in size and adequate for majority of cases.

#! Deleting website elements
* Go to the end of the page where the elementor card is present, right click on it and click on the "Delete All Content" button.

#! Adding Sections
* Click on the "+" icon, then select "Flexbox" and then direction of that container.

#! Adding Background Image
1. Select the "Style" section from the "Edit Container" option.
2. Go to Background, then select Background type as "Classic" and you will get the option to add a background image.
3. Now go to the layout section and set a "Min Height" of that container.
4. Again go to "Style" section, in "Background", set Display Size as "Cover" 

#! Adding Gradient to the Image
5. Now in Background Overlay, select Background Type as "Gradient". 
6. Change the "Color" to the left most side of the bar i.e white
7. Change the "Second Color" to the right most side of the bar i.e black
8. Change the "Location" to 80 and Opacity to 0.75.
9. Explore the other options and make changes as per your requirements.

#! Adding Text
1. Click the "+" icon, then select "Heading" and then select the "Content" section. Type down the required text.
2. Now go to the "Style" section and select "Text Color" and change the color to white.
3. For SEO purposes, always have some H1 tags as heading.
4. Now again click the "+" icon and drag drop a "text editor" beneath the heading text.
5. Generate some professional text for this about pafge from any free AI generative tool like Google Gemini.
6. Set text color to white.

#! Changing position of elements in a section
1. Right click on the container and select "Edit Container"
2. Go to "Layout" and set justify content to "End"
3. Now change Content Width from "Full Width" to "Boxed" and set width to around 700px
4. Now click on the heading and the text below and set their alignment to "center"

#! To add space between elements
1. Search for a widget called "Spacer" and drag drop it to the required area.
2. Change the space property of the spacer to 30px as per the requirement.

#! Adding color boxes
1. Click on the "+" icon, then select "Flexbox" and then direction of that container.
2. In the "Edit Container", go to "Style" and select "Background" to "white".
3. Now add another container inside this container.
4. Now using the chrome extension "Color Zilla", get the color code of the box from the inspirational page.

5. We can copy paste elements from one container to another.
6. Copy paste the heading from the first container to the second container
7. Set it as H2 tag and change the color to global default color by clicking the anti clockwise button inside the text color option
8. Change text to "Our Vision"

9. Now copy paste the text paragraph from the first container to the second container.
10. Make changes as before like resetting the text color to global default color, etc.
11. Get the text for our vision from Gemini.

#! Using Page Navigator/ Structure in Elementor
1. Click on the Burger Menu like icon on the top left side of the screen.
2. Click on the element you want to work on without any hassle.
3. Select the first container and set padding to 120px 10px 120px 10px as per the requirement.
4. Select the second container and set padding to 80px 80px 80px 80px as per the requirement.
5. Select the Style tab and set border radius to 30px.

#! Next Section
1. Click on the "+" icon, then select "Flexbox" and then direction of that container.
2. Set color of that container to black and padding to 120px 10px 120px 10px.

#! Next Section
1. Click on the "+" icon, then select "Flexbox" and then set the 3rd direction i.e 2 column in one row for that container.
2. Now set a background image by downloading a relevant image from unsplash.com.
3. Change display size to "Cover".
4. Now increase the height of the container by either setting the "Min Height" to 800px or by setting padding to 240px 10px 240px 10px by inlinking the padding first.
5. Copy paste the heading to the first division and change the text.
6. Duplicate the heading in the first division by right clicking the heading and selecting "Duplicate". Set it to H6 tag.
7. Get a button using drag drop and place it below the heading.
8. Change the text and set the link url to contact page by typing "contact"
9. To read the text better, we need to add a background overlay.
10. To to edit container for the primary container and select the Style tag.
11. Select the "Background Overlay" and select Classic, and set color to black.

#! Editing Services Page
1. Click on the Services page and click "Edit with Elementor".
2. Add a button widget below the first card and change its text to "Learn More". Leave the link empty for now.
3. Now right click on that button and select "Copy" and paste it below text of every card.
4. Delete the "Book a Consultation" button by right clicking on the button and selecting delete.
5. Now we can observe that the "Learn More" butttons in each card is not aligned properly. 
6. To fix this, go to the "Structure" option and select the container for the card. Now set "Justify Content" to "Space Between". Do the same for the rest of the cards.
#? Trick
7. Another approach to do this is to set the "Justify Content" to "Space Between" for the first container and then right click on the container and select "Copy" and then right click on the other containers and select "Paste Style".

#! Adding Subpages
1. Go to the backend or the wp-admin and go to pages. Click on Add New Page
2. Turn off the "Full Screen Mode" by clicking on the 3 dots on the top right corner of the page.
3. Now click on "Page Attributes" and set "Parent" to Services.
4. To to the backend and the Pages tab, you will now be able to see the newly added subpage with a prefix of "-"

#! Importing New Page Templates
1. Edit the "Drain Cleaning & Repair" page and click on "Edit with Elementor".
2. Now click the "Starter Templates" option.
3. Select the Pages option to get the entire page but beside it lies a option of Blocks which allows us to add individual blocks to the page.
4. Select a page from the given options and click on Import Template. (Roofing Agency - About Page)
5. Now since we don't like the header of the page, we can delete that container. Now open a new tab and go to the services page and click edit with elementor.
6. Copy the header of the services page by clicking on the header and select "Copy" and then go back to the subbpage and add a new container and right click on it and select "Paste".
7. Make changes to the header section and delete the containers from the page which we do not need.

#! Duplicating Pages
1. Install a new plugin called "Duplicate Page".
2. Now go to the pages tab and below each page, you will get a new option called "duplicate this".
3. Duplicate the new subpage 6 times for 6 different cards.
4. Now to edit these 6 subpages, rather than clicking the edit option, use the quick edit option and change the title and other details as needed. By default, these pages have status as "Draft", change it to published.
5. Copy paste the titles of the cards to these subpages title.
6. Now go the the services page and edit with elementor. Set the respective links to the respective buttons for the 6 cards used.
7. Make changes to these subpages to make it unique for each card as now we just have duplicates of "Drain Cleaning & Repair" page.

#! Adding Blog Page
1. Adding a blog page with trending questions and answers posts will help gain trust of customers that you are a expert in your field.
2. Another big benifit of having blog on your website is that you get SEO(Search Engine Optimization) benifits from it.
3. SEO => The process of improving your sites visibility in search engines to get more organic traffic to your website.

4. Go to pages and create a new page and name it as "Blog".
5. Now go to the wp-admin and hover on settings and click on "Reading Settings".
6. Set Posts page to "Blog" and click save.

#! Designing Blog Page
1. To understand the default design of the blog page, we need some more blogs. So either we can add blogs manually or we can use a plugin to add dummy blog posts.
2. To generate fake content, install plugin called "FakerPress".
3. On the left side bar, you can now see a plugin called "FakerPress". Hover over it and click on "Posts".
4. Set Quantity field to 12 to 24 and click generate.
5. Now go to the blog page and click Customize.
6. Select Blog option and then "Blog/Archive". Select the GENERAL tab.
7. Now from the available options, go to "BLOG LAYOUT" and select the "Cover" layout.
8. Go to POSTS STRUCTURE to set Post Elements and Meta accordingly.
9. Set post per page to 12 so that we have even rows.
10. Set Background Overlay to Blue color and reduce the opacity to 30%.

#! Adding Custom CSS code
1. Use "Ctrl+Shift+I" to open the inspect mode in Chrome.
2. Hover on the div which covers the entire 1st card and set its height from 100% to 400px.
3. Now copy that code snippet as we will use it as custom CSS.
"
.ast-separate-container .ast-blog-layout-6-grid .ast-article-inner, .ast-plain-container .ast-blog-layout-6-grid .ast-article-inner {
    height: 400px;
}
"
4. Now go back to the main menu where the Additional CSS option is available at the end.
5. Click on the "Additional CSS" option and paste the code snippet.
6. Click on the publish button.

#! Designing Blog Post
1. Now go to the blog page and click on the first post.
2. Click on customize, then blog, then "Single Post".
Now I want the title of the blog post inside the image.
3. Click on "Post Title Area" and select "Layout 2" from the Banner Layout.
4. Set Container Width to "Full Width" and in the Structure option, enable the "Use as background". Due to a bug in Wordpress, you will not be able to see the changes immediately. Also change the overlay color and its opacity according to your needs. You will have to disable and enable the "Featured Image" option to see the changes.
5. Now go to the Design tab and select the Title Color, Text Color, Link Color to white. Again you will have to disable and enable the "Featured Image" option to see the changes.
6. Now go to the General tab again and disable Comments and Taxonomies and enable Author and Date under Meta section.
7. In Author, change the Prefix label and enable the Author Avatar.
8. Now go back to the Singe Post General Post and under Sidebar Layout, select the "Right Sidebar" option. This was just for trial, revert it back to default.
9. Publish the changes.

#! Building Site Header

2. Click on Header Builder.
3. We have 2 tabs, "Elements" and "Design"
4. You will get lots of options in the left panel but the most useful one is the elongated bar on the main screen which for this theme consists of "Above Header", "Primary Header" and "Below Header".
5. The primary header consists of: "Site Title & Logo", "Primary Menu", "HTML 2", "Button".
6. You can drag drop and set the placement of the elements as you wish. I deleted the "HTML 2" element and set the "Primary Menu" to center.
7. Click onn the top right portion from the "Above Header" and add "Social".
8. Now click on the social icons menu and make changes to the icons. Go to the Design tab and make changes to the icons as you wish.
9. Click on the Primary Menu and select "Configure Menu From Here".
10. The "Off-Canvas Menu" is by default set same as the "Primary Menu" but it is for mobile version. When in mobile version, you will see a hamburger icon at the top right corner. Click on it to get the menu in the mobile version.
